# Last Delivery: Courier Rush 3D

Game 3D **"Last Delivery: Courier Rush"** — seorang kurir mengantar paket keliling kota sambil menghindari mobil, palang penghalang, dan lubang jalan. Dibuat untuk tugas mata kuliah **Teknologi Game**.

## Teknologi
- **Unity 6** (URP)
- **C#** (New Input System)
- **100% dari kode + bentuk dasar (primitif)** — tanpa asset/model jadi. Tekstur (rumput, aspal, bata, genteng, hati, garis finish) dibuat lewat kode.

## Cara Main
- **Menu**: klik **START** untuk mulai, **QUIT** untuk keluar.
- **Scene 1 (Pos Kurir)**: jalan (WASD) masuk pos, ambil paket.
- **Scene 2 (Jalan Raya)**: lari otomatis, **A/D** geser lajur, **Spasi** lompat. Hindari mobil & palang, ambil ❤️ untuk nyawa, lewati **Bos Truk**, capai garis finish sebelum waktu habis.

## Struktur
- `Assets/Scripts/` — semua kode game (Player, Enemy, Level, Items, UI, Core)
- `Assets/Scenes/` — Menu, Level1Scene1, Level1Scene2
- `Assets/Materials/`, `Assets/Textures/` — material & tekstur hasil kode
